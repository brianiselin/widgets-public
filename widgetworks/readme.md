# Widget Works

Widget Works is a collection of custom HTML, CSS, and JavaScript widgets and visualizations designed for embedding in articles, websites, and digital projects.

## Overview

This repo contains interactive data visualization components and utilities that I develop to enhance user engagement and data storytelling. These widgets are optimized for ease of integration and performance.

## Features

- Responsive and interactive visualizations  
- Modular and reusable code components  
- Compatibility with multiple platforms and publications  

## Usage

Clone this repository to access the widgets. Each folder contains the relevant assets and instructions for embedding or customization.
