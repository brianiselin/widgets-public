# RentCheckR — Mini Brand Kit (v2)

## Palette
- Deep navy: **#1e3a5f**
- Soft teal: **#4a9b9b**
- Warm cream: **#faf8f5**
- Alert orange: **#e67e22**

## Suggested usage
Use **navy** for primary text + trust cues. Use **teal** for interactive elements (buttons/links). Use **orange** sparingly for alerts (“possible overcharge”) and CTA accents.

## CSS variables
```css
:root {
  --rc-navy: #1e3a5f;
  --rc-teal: #4a9b9b;
  --rc-cream: #faf8f5;
  --rc-orange: #e67e22;
}
```

## Logo usage
- Prefer the **mark-only** icon at very small sizes (favicons).
- Prefer the **horizontal lockup** in headers/nav; **v2 increases spacing** between mark and wordmark for better readability.
- Prefer the **named square icon** for store listing graphics and larger UI placements.

## Clear space
Keep clear space around the lockup at least **1/3 of the mark height**.
