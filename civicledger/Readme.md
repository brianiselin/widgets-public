# The Civic Ledger

This folder contains HTML visualizations and interactive widgets created specifically for The Civic Ledger Medium publication.

These visualizations are designed to complement and enhance articles published in The Civic Ledger, providing readers with dynamic data presentations and insights.

## Contents

- Interactive charts and graphs  
- Data-driven widgets  
- Visual storytelling elements  

## Usage

The visualizations here are intended to be embedded or linked within Medium articles for enhanced reader engagement. They are developed using HTML, CSS, and JavaScript.
